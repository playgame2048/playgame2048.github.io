      📘 How to Play Crossword Mini

      Basic Rules

      Fill the 7×7 grid so every row and column forms a word. Numbers indicate start of a word. Use the clues on the left.

      • One letter per white cell
      • No skipping cells
      • All words are valid

      Pro Techniques

      Use crossing letters to your advantage. Start with words you know, then use the intersecting letters to guess others.

      • Look for common prefixes
      • Vowels appear often
      • Check word length

      Smart Tips

      The highlighted row/column helps you focus. If a cell turns red, double‑check the clue. Practice daily to improve speed.

      • Solve with pencil (mentally)
      • Use process of elimination
      • Keep a steady pace
