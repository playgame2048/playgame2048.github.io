      How to play Crossword Mini

      📋 Basic rules

      Fill the 7x7 grid so every row and column forms a valid word. Each white cell accepts one letter. Numbers show where a word starts. Use the clues on the right.

      ⚡ Tips to solve faster

      Start with words you’re certain about. Use crossing letters to eliminate options. The highlighted row/col helps you focus on one word at a time.

      🧩 Pro techniques

      Look for high‑frequency letters (E, T, A). If a cell turns red, double‑check the clue. The puzzle is solved when all cells are green-free (no red).
